# UI-Events
#### IDs
后端仅基于ID名称获取组件，无法基于不同布局文件区分组件ID，也无法区分组件类型。调用到不是当前布局上组件，运行时才能发现错误。
因此，ID的名称必须能够体现具体的完整的信息，从而避免出错。
较好的命名方法，布局类型_布局名称_组件类型_组件名称
